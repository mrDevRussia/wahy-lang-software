## 📖 نظرة عامة

**Wahy وَحي** هي لغة برمجة عربية مبسطة مصممة خصيصاً لتعلم أساسيات برمجة الويب. تتيح للمطورين المبتدئين إنشاء صفحات HTML باستخدام أوامر بسيطة وواضحة باللغة العربية الفصحى.

### 🎯 الهدف من المشروع

- **التعليم**: جعل تعلم البرمجة أكثر سهولة للمتحدثين بالعربية
- **البساطة**: أوامر واضحة ومفهومة بدون تعقيد
- **الإنتاجية**: إنتاج صفحات HTML جاهزة للنشر فوراً
- **الإلهام**: تشجيع المطورين العرب على دخول عالم البرمجة

## ✨ المميزات الرئيسية

### 🌐 واجهات متعددة
- **واجهة سطح المكتب**: محرر متقدم مع معاينة مباشرة
- **واجهة الهاتف المحمول**: تصميم متجاوب ومحسن للأجهزة المحمولة
- **واجهة مصادقة**: نظام تسجيل دخول وإدارة حسابات المستخدمين

### 💻 أدوات التطوير
- **محرر كود متقدم**: مع تمييز بناء الجملة للأوامر العربية
- **معاينة مباشرة**: ترى النتيجة فور كتابة الكود
- **نظام إدارة الملفات**: حفظ وتنظيم مشاريعك
- **تحميل HTML**: إنتاج ملفات جاهزة للنشر

### 📚 دعم التعلم
- **دليل تعليمي تفاعلي**: خطوات مبسطة لتعلم اللغة
- **مرجع الأوامر**: دليل شامل لجميع الأوامر المتاحة
- **أمثلة عملية**: نماذج جاهزة للتعلم والتطبيق

### 🏗️ البنية التقنية
- **مفسر JavaScript**: أداء سريع وموثوق
- **قاعدة بيانات PostgreSQL**: تخزين آمن للملفات والحسابات
- **واجهة React**: تفاعلية وعصرية
- **خادم Express**: مستقر وقابل للتوسع

## 🚀 كيفية التثبيت

### متطلبات النظام
- Node.js (الإصدار 18 أو أحدث)
- PostgreSQL (قاعدة البيانات)
- Git

### خطوات التثبيت

1. **استنساخ المستودع**
```bash
git clone https://github.com/username/wahy-lang.git
cd wahy-lang
```

2. **تثبيت التبعيات**
```bash
npm install
```

3. **إعداد قاعدة البيانات**
```bash
# تأكد من وجود PostgreSQL
# قم بإنشاء قاعدة بيانات جديدة
createdb wahy_db

# إعداد متغيرات البيئة
cp .env.example .env
# قم بتحرير ملف .env وإضافة رابط قاعدة البيانات
```

4. **تهيئة قاعدة البيانات**
```bash
npm run db:push
```

5. **بدء التطبيق**
```bash
npm run dev
```

6. **فتح المتصفح**
افتح `http://localhost:5000` في متصفحك

## 📝 دليل استخدام لغة وحي

### البنية الأساسية

كل برنامج في لغة وحي يجب أن يبدأ وينتهي بهذين الأمرين:

```wahy
افتح صفحة "عنوان الصفحة"

# محتوى الصفحة هنا

أغلق صفحة
```

### 📚 دليل الأوامر الشامل

#### 📄 أوامر الصفحة الأساسية

| الأمر | الوصف | المثال |
|-------|--------|---------|
| `افتح صفحة "العنوان"` | بدء صفحة HTML جديدة | `افتح صفحة "موقعي الأول"` |
| `أغلق صفحة` | إنهاء الصفحة | `أغلق صفحة` |

#### ✏️ أوامر المحتوى

| الأمر | الوصف | المثال |
|-------|--------|---------|
| `أضف عنوان "النص"` | إضافة عنوان رئيسي (H1) | `أضف عنوان "مرحباً بالعالم"` |
| `أضف عنوان_فرعي "النص"` | إضافة عنوان فرعي (H2) | `أضف عنوان_فرعي "قسم جديد"` |
| `أضف فقرة "النص"` | إضافة فقرة نصية | `أضف فقرة "هذا نص الفقرة"` |
| `أضف رابط "النص" "الرابط"` | إضافة رابط | `أضف رابط "موقع جوجل" "https://google.com"` |
| `أضف صورة "الرابط" "الوصف"` | إضافة صورة | `أضف صورة "image.jpg" "وصف الصورة"` |

#### 🎨 أوامر التنسيق والتصميم

| الأمر | الوصف | المثال |
|-------|--------|---------|
| `غيّر لون_الخلفية إلى "اللون"` | تغيير لون خلفية الصفحة | `غيّر لون_الخلفية إلى "lightblue"` |
| `غيّر لون_النص إلى "اللون"` | تغيير لون النص | `غيّر لون_النص إلى "darkblue"` |
| `غيّر الخط إلى "نوع_الخط"` | تغيير نوع الخط | `غيّر الخط إلى "Arial"` |

#### 📋 أوامر القوائم

| الأمر | الوصف | المثال |
|-------|--------|---------|
| `ابدأ قائمة` | بدء قائمة نقطية | `ابدأ قائمة` |
| `أضف عنصر "النص"` | إضافة عنصر للقائمة | `أضف عنصر "العنصر الأول"` |
| `أنهِ قائمة` | إنهاء القائمة النقطية | `أنهِ قائمة` |
| `ابدأ قائمة_مرقمة` | بدء قائمة مرقمة | `ابدأ قائمة_مرقمة` |
| `أنهِ قائمة_مرقمة` | إنهاء القائمة المرقمة | `أنهِ قائمة_مرقمة` |

#### 🏗️ أوامر التخطيط

| الأمر | الوصف | المثال |
|-------|--------|---------|
| `ابدأ قسم` | بدء قسم جديد (div) | `ابدأ قسم` |
| `أنهِ قسم` | إنهاء القسم | `أنهِ قسم` |
| `أضف خط_فاصل` | إضافة خط أفقي | `أضف خط_فاصل` |
| `أضف مسافة` | إضافة مسافة فارغة | `أضف مسافة` |

## 🛠️ التطوير والمساهمة

### بنية المشروع

```
wahy-lang/
├── client/                 # الواجهة الأمامية (React)
│   ├── src/
│   │   ├── components/    # المكونات المشتركة
│   │   ├── pages/         # صفحات التطبيق
│   │   ├── hooks/         # Custom hooks
│   │   └── lib/           # المكتبات المساعدة
├── server/                # الخادم الخلفي (Express)
│   ├── routes.ts          # مسارات API
│   ├── storage.ts         # طبقة قاعدة البيانات
│   └── wahy-interpreter.ts # مفسر لغة وحي
├── shared/                # الأنواع المشتركة
│   └── schema.ts          # مخططات قاعدة البيانات
└── docs/                  # الوثائق
```

## 🐛 استكشاف الأخطاء وإصلاحها

### مشاكل شائعة

**Q: لا يعمل المفسر/تظهر رسالة خطأ**
```
A: تأكد من:
- وجود قاعدة البيانات وتشغيلها
- صحة رابط قاعدة البيانات في ملف .env
- تشغيل الأمر npm run db:push
```

**Q: الواجهة لا تظهر بالعربية بشكل صحيح**
```
A: تأكد من:
- دعم متصفحك للغة العربية
- تفعيل الاتجاه RTL في CSS
- وجود خطوط عربية مناسبة
```

**Q: بطء في تحميل الملفات**
```
A: قد يكون بسبب:
- اتصال قاعدة البيانات
- حجم الملفات المحفوظة
- إعدادات الخادم
```
